# Task Management System

A simple Task Management web application built using Python and Django. This project allows users to create, update, complete, and delete tasks through a clean and user-friendly interface.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- View all tasks in one place
- Track task creation date and time

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite

## Project Structure

- Task creation and management
- Django Models and Views
- Templates for frontend rendering
- SQLite database integration
